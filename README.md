# 🛒 Java Shopping Cart Application

A complete, standalone desktop shopping cart application built with Java Swing. Features a modern GUI, real-time cart management, professional receipt generation, and input validation. Perfect for learning Java GUI development or as a foundation for point-of-sale systems.

**Key Features:**
- Real-time cart updates with automatic total calculation
- Professional receipt generation dialog
- Input validation and error handling
- Duplicate item merging
- Clean, modern UI with custom styling
- Menu bar with file operations

**Tech Stack:** Java 8+, Swing, AWT

**Use Cases:** Retail POS systems, e-learning Java projects, small business inventory management
